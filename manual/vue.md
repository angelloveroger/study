# vue

> ## 一.vue运行
>> ***1.node.js环境（npm包管理）即node.js的下载和安装***
>>> *①.node版本查看*  
>>>
>>>     node -v
>>> *②.npm版本查看*
>>>
>>>     npm -v
>>>
>> ***2.npm淘宝镜像cnpm的安装***
>>>
>>>     npm install -g cnpm --registry=http://registry.npm.taobao.org
>>>
>> ***3.vue-cli脚手架构建工具安装***
>>>
>>>     cnpm install vue-cli
>>>
>> ***4.创建一个基于webpack的名为fiistVue的项目***
>>>
>>>     vue init webpack firstVue
>>>
>> ***5.安装项目依赖***
>>>
>>>     cnpm install
>>>
>> ***6.运行项目***
>>>
>>>     npm run dev
