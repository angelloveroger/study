数据存储

问题1：为什么使用计算机？
	存储数据，计算数据

问题2：数据存储在哪里？
	数据存储在内存

问题3：数据怎么在内存存储的？
	内存中以二进制形式存储数据	
	存储单位 1T = 1024G
			 1G = 1024M
			 1M = 1024K	
			 1K = 1024字节
			 1字节 = 8bit	
	数据存储：数据存储的最小单位是字节。计算机存储数据，先开辟内存空间，然后存储数据。计算机开辟内存空间的最小单位是字节。在存储数据时，用最高位标识符号，1表是负数，0表是正数

问题4：进制之间转换
十进制转二进制 
八进制转二进制   一位变三位，按十进制转换
十六进制转二进制 一位变思维，按十进制转换
